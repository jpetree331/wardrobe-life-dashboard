# Wardrobe — life-dashboard workspace

A personal mind-palace web app for contemplative practice. The umbrella name
is *Wardrobe* (after Narnia's): the entrance looks ordinary, the inside is a
country.

## Layout

| Path | What's there |
|---|---|
| [`app/`](app/) | The running React + Vite + Supabase application. **Start here.** |
| [`Life Board/design_handoff_life_board/`](Life%20Board/design_handoff_life_board/) | Original Sanctuary + Hallway visual handoff — the design source of truth for those rooms. |
| [`Timeline/timeline_handoff/`](Timeline/timeline_handoff/) | Timeline visual handoff — design source of truth for the Timeline tab. |
| [`docs/`](docs/) | Loose notes and pre-build ideation. |

## Getting started

See [`app/README.md`](app/README.md) for setup, environment variables, the
two SQL migrations to run in Supabase, and the deploy steps for Vercel.
